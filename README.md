# Introduction

This is a [JHipster](https://www.jhipster.tech/) blueprint, that is meant to be used in a JHipster application.

# Prerequisites

As this is a [JHipster](https://www.jhipster.tech/) blueprint, we expect you have JHipster basic knowledge:

- [JHipster](https://www.jhipster.tech/)

# Installation

To install or update this blueprint:

```bash
npm install -g generator-jhipster-generator-jhipster-primeng-blueprint
```

# Usage

To use this blueprint, run the below command

```bash
jhipster --blueprints generator-jhipster-primeng-blueprint
```

You can look for updated generator-jhipster-primeng-blueprint blueprint specific options by running

```bash
jhipster app --blueprints generator-jhipster-primeng-blueprint --help
```

And looking for `(blueprint option: generator-jhipster-primeng-blueprint)` like

## Pre-release

To use an unreleased version, install it using git.

```bash
npm install -g jhipster/generator-jhipster-generator-jhipster-primeng-blueprint#main
jhipster --blueprints generator-jhipster-primeng-blueprint --skip-jhipster-dependencies
```
